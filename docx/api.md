# 1.1 Login API
## Request
- Method: POST
- Endpoint: /login
- Header: 
    - Content-Type: application/json
    - apikey: string
- Request Body:
    "application.code": string,
    "user.email": string,
    "user.password": string,
    "device": string"
## Response
- Status Code: 200 OK
- Response Body:
    - code: string
    - partners: array of objects
        - code: string
    - access_token: string
